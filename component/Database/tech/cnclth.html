2. M-Code

Code Label Function
M00 Program Stop(프로그램 정지)
M01 Optional Program Stop(조건부 프로그램 정지)
M02 Program End(프로그램 종료)
M03 Spindle Rotation CW(주축 정회전)
M04 Spindle Rotation CCW(주축 역회전)
M05 Spindle Stop(주축 정지)
M08 Coolant ON(절삭유 ON)
M09 Coolant Off(절삭유 Off)
M30 Program Restart(프로그램 재실행)
M98 Sub Program Call(보조프로그램 실행)
M99 Sub Program End(보조프로그램 종료)
