※ 코드값 구하는 방법

1. 초기값 AC00
2. 종성(받침)의 번호값 가산
3. 중성(모음)의 값 * 28의 값을 가산 (28은 종성 종류의 개수)
4. 초성(자음)의 값 * 28 * 21의 값을 가산 (21은 중성의 종류의 개수)
ex) "돌"의 유니코드값
① -> ㄷ,ㅗ,ㄹ 의 번호 = 3, 8, 8
② -> 3*21*28 + 8*28 + 8 = 1764 + 224 + 8 = 1996 = 7cc(16진수)
③ -> AC00 + 7CC = B3CC
